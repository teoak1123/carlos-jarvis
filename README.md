# 🤖 Carlos Jarvis

Carlos Jarvis es un bot avanzado para Discord diseñado para centralizar moderación, economía, juegos, utilidades, herramientas de Minecraft, sistemas sociales y funciones de participación comunitaria en un solo proyecto.

La filosofía del bot es simple:

> Menos bots, más funciones, más actividad y una mejor experiencia para la comunidad.

---

# ✨ Características

## 🛡 Moderación avanzada

Sistema completo de moderación para servidores de cualquier tamaño.

### Funciones

- Advertencias
- Silencios temporales
- Silencios de voz
- Expulsiones
- Baneos
- Baneo falso (broma)
- Lista negra
- Lista negra temporal
- Historial de sanciones
- Apelaciones
- Logs de moderación

---

## 🚨 AutoModeración

Protección automática configurable.

### Detecta

- Spam
- Flood
- Mensajes repetidos
- Menciones masivas
- @everyone
- @here
- Invitaciones de Discord
- Links sospechosos
- Copypastas
- Mensajes excesivamente largos
- Exceso de líneas
- Publicidad
- Contenido no permitido

### Acciones automáticas

- Eliminar mensaje
- Advertir usuario
- Silenciar temporalmente
- Registrar evento
- Notificar al canal de moderación

---

## 🔥 AntiRaid

Protección contra ataques masivos.

### Detecta

- Entradas masivas
- Creación masiva de cuentas
- Spam coordinado
- Menciones masivas
- Ataques automatizados

---

## 💣 AntiNuke

Protección contra destrucción del servidor.

### Detecta

- Creación masiva de canales
- Eliminación masiva de canales
- Creación masiva de roles
- Eliminación masiva de roles
- Modificaciones peligrosas

# 🎯 Objetivo

Carlos Jarvis busca ofrecer en un único bot:

- Moderación avanzada
- Protección del servidor
- Economía competitiva
- Juegos comunitarios
- Herramientas útiles
- Participación constante
- Sistemas sociales
- Funciones para Minecraft

Todo integrado en una sola experiencia.

---

# ❤️ Hecho para comunidades activas

Carlos Jarvis está pensado para servidores que buscan mantener a sus miembros participando, jugando y colaborando todos los días sin depender de múltiples bots externos.
